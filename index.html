New String Expression Evaluator

By: Pearl Lin

The new string expression evaluator works with functions in addition to PEMDAS expressions! This was made possible by utilizing the JavaScript eval() function as opposed to parsing the string with hard code. Parentheses can now be used to denote multiplication!

Guidelines for Inputting Your Expressions

✔ Please use * · () × to denote multiplication. Reserve the letter x for representing variables.


✔ Use either / or ÷ to indicate division


✔ Use the carrot symbol ^ to represent exponentiation.


✔ Use the + symbol for addition and use any of the following symbols: - ˗ − ﹣ － for subtraction & negative numbers


✔ Supported ways of writing parentheses: () [] {}


✔ Remember to pair all the sets of parentheses you write! If the number of opening parentheses and closing parentheses does not equal, an error will show


✔ Please provide the function evaluator with a numerical value of the variable x. Both the PEMDAS expression evaluator and the function expression evaluator are programmed to give you an exact numerical result, without variables.
